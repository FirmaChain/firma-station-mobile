### 1. Requirements

-   Node.JS: v16.20.2
-   OpenJDK: 11.0.24

### 2. install

> **Do not remove .lock files for no reason.**

1. Create a project with git clone.

2. Install modules.

```bash
npm install
```

4. Install pod modules for iOS

```bash
cd ios
pod install
cd ..
```

### 3. Run

```bash
npm run ios
npm run android
```
